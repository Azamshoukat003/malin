# MalinKiddy Frontend

Next.js 14 App Router frontend for `audio.malinkiddy.com`.

## Setup

1. Copy `.env.example` to `.env.local`.
2. Install:
   - `npm install`
3. Start dev server:
   - `npm run dev`
4. Build production:
   - `npm run build`
   - `npm run start`

## Required Env

- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_APP_URL`
